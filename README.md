# Практична робота 7.1
## Варіант 4 — Бібліотека книг

### Опис
Реалізовано систему управління колекцією книг.

### Функціонал:
- Каталог книг (назва, автор, рік, жанр, сторінки, статус)
- Пошук за назвою, автором, жанром, роком
- Фільтри (прочитані/непрочитані, жанр, роки)
- Статистика (кількість книг, сторінки, топ автори)
- Рекомендації книг

### Використані методи:
filter(), map(), reduce(), sort()

### Як запустити:
Відкрити index.html у браузері
